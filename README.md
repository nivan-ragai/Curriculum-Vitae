## المهارات التقنية والإدارية

**جداول البيانات المتقدمة:** إعداد جداول محورية، استخدام دوال البحث والجمع الشرطي المتقدمة (مثل VLOOKUP, SUMIFS)، وتصميم تقارير مالية ديناميكية.

**معالجة النصوص:** إنشاء وتنسيق التقارير والمراسلات الرسمية باحترافية.

**الفاتورة الإلكترونية (E-Invoicing):** خبرة عملية في تطبيق المنظومة.

**الرقابة المالية على المخزون والتكاليف:** تحليل انحرافات الجرد، تسويات الأرصدة، والمساهمة في إعداد الموازنات التقديرية للحد من الهدر.

**إدخال البيانات:** سرعة ودقة عاليتان في إدخال وتدقيق البيانات المالية.

**إدارة الوقت والعمل الجماعي:** القدرة على تحديد الأولويات والعمل بروح الفريق لتحقيق الأهداف المشتركة.

**تنظيم الملفات والبيانات الرقمية:** أرشفة إلكترونية منظمة تضمن سهولة الوصول للمعلومات والمراجعة.
